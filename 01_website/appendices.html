Deployment Manual

  • Set up Kinects as in ideal room plan at specific distances, with each Kinect connected to a computer, where each computer needs a copy of the project and has Unity installed. Please note that since we currently only have access to two Kinects, currently the Prototype is likely to only work with 2 Kinects set up in the Kinect 1 and Kinect 3 positions.

  • On each computer, load the project in Unity and run the project. Press Start, and then Watch Playback. To record a dance, say “Record” (loudly) and the Kinects should begin to record. Once you wish the recording to finish, say “Stop”.

  • This will create a .txt file on each computer connected to a running Kinect. At the moment the .txt files need to be manually moved to whichever computer is the master computer.

  • On the master computer, press the Merge button and select the .txt files you wish to merge. This will create a master .txt file that is the combined skeletal data of the recording.

  • Go back into Watch Playback, and click once to begin playback. (currently a skeleton of the recording will be shown) You can zoom with the scroll wheel and rotate with the arrow keys as the playback is going on
